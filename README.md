# 🏥 Hospital Operations & Patient Insights Dashboard
 📌 Project Overview
This Power BI dashboard provides a comprehensive analysis of hospital operations and patient-related metrics using healthcare data.
The dashboard enables healthcare administrators and analysts to monitor patient flow, departmental performance, revenue generation, readmission rates, and patient demographics through interactive visualizations.
## Business Problem
Hospitals generate large volumes of operational data daily. Without proper analysis, it becomes difficult to:
- Monitor patient admissions and visits
- Track departmental performance
- Analyze revenue contribution
- Identify high readmission rates
- Understand patient demographics
- Improve resource allocation

This dashboard transforms raw healthcare data into actionable business insights.

## 📊 Key Metrics

- Total Patient Visits
- Average Length of Stay (LOS)
- Total Revenue Generated
- Inpatient vs Outpatient Distribution
- Monthly Patient Visit Trends
- Department-wise Revenue Analysis
- 30-Day Readmission Rate
- Patient Age Group Distribution

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- Excel / CSV Dataset
